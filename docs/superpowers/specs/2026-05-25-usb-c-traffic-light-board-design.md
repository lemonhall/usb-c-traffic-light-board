# USB-C 红绿灯控制板设计

## 目的

做一块完整替换原红绿灯小板的新 PCB。新板通过 USB-C 供电，并通过 USB CDC 虚拟串口接受电脑控制。

## 机械范围

第一版板框使用 `57mm x 14mm` 矩形。三颗灯的位置先按照片大致竖向排列，等你提供卡尺尺寸后再精修。板上需要容纳三颗 LED、USB-C 母座、CH552G 主控、USB 阻容、LED 限流电阻和测试焊盘。

## 电气范围

主控使用带原生 USB 的 CH552G，而不是 ESP32，因为本需求不需要 Wi-Fi 或蓝牙。USB-C 提供 `5V` 供电以及 USB2.0 的 `D+ / D-` 数据线。`CC1 / CC2` 使用外置 `5.1k` 下拉电阻声明受电设备。每颗 LED 由一个 GPIO 通过限流电阻独立控制。

## 控制协议

电脑通过虚拟串口发送 ASCII 文本命令，例如 `R 1`、`Y 0`、`G 1`、`A 0`、`S 1 0 1` 和 `?`。板子返回 `OK`、当前状态或解析错误。

## 风险

- CH552G 的物理引脚必须在最终 KiCad 符号和封装选定后再次核对。
- USB-C 母座封装必须和最终采购型号一致。
- 三颗灯中心、安装孔和缺口需要精确尺寸后才能打样。
- 固件需要先选定一个稳定的 CH55x CDC 示例工程。
