# 引脚和网络分配草案

## USB-C

| 信号 | 连接位置 | 说明 |
| --- | --- | --- |
| USB_C_VBUS | 主控电源 / 板上 `5V` | C1 靠近 USB-C 插座 |
| USB_C_GND | 板上地 | USB-C 外壳附近多打地过孔 |
| USB_D_PLUS | 通过 R4 接主控 USB D+ | 走线尽量短，并和 D- 成对 |
| USB_D_MINUS | 通过 R5 接主控 USB D- | 走线尽量短，并和 D+ 成对 |
| USB_CC1 | `5.1k` 到地 | 声明自己是受电设备 |
| USB_CC2 | `5.1k` 到地 | 声明自己是受电设备 |

## 三路灯控制

最终物理引脚必须等 KiCad 符号和 CH552G 封装确认后再锁死。

| 功能 | 网络名 | 暂定主控端口 | 驱动方式 |
| --- | --- | --- | --- |
| 红灯 | LED_R | P1.4 | GPIO 输出高电平点亮 |
| 黄灯 | LED_Y | P1.5 | GPIO 输出高电平点亮 |
| 绿灯 | LED_G | P1.6 | GPIO 输出高电平点亮 |

## 测试焊盘

| 焊盘 | 网络名 | 用途 |
| --- | --- | --- |
| TP1 | 5V | 量供电 |
| TP2 | GND | 接地夹 |
| TP3 | USB_D_PLUS | USB 调试 |
| TP4 | USB_D_MINUS | USB 调试 |
| TP5 | RST | 复位 / 烧录辅助 |
| TP6 | LED_R | 红灯信号探针 |
| TP7 | LED_Y | 黄灯信号探针 |
| TP8 | LED_G | 绿灯信号探针 |
