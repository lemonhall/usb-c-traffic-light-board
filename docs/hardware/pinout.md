# 引脚和网络分配草案

## USB-C

| 信号 | 连接位置 | 说明 |
| --- | --- | --- |
| USB_C_VBUS | 主控电源 / 板上 `5V` | C1 靠近 USB-C 插座 |
| USB_C_GND | 板上地 | USB-C 外壳附近多打地过孔 |
| USB_D_PLUS | 通过 R4 接主控 USB D+ | 走线尽量短，并和 D- 成对 |
| USB_D_MINUS | 通过 R5 接主控 USB D- | 走线尽量短，并和 D+ 成对 |
| USB_CC1 | `5.1k` 到地 | 声明自己是受电设备 |
| USB_CC2 | `5.1k` 到地 | 声明自己是受电设备 |

## 三路灯控制

以下按 CH552G 的 SOP-16 引脚整理。下单前仍要对照最终采购型号的数据手册和封装丝印再核一次。

| 功能 | 网络名 | 主控物理脚 | 主控端口 | 驱动方式 |
| --- | --- | --- | --- | --- |
| 红灯 | LED_R | 2 脚 | P1.4 | GPIO 输出高电平点亮 |
| 黄灯 | LED_Y | 3 脚 | P1.5 | GPIO 输出高电平点亮 |
| 绿灯 | LED_G | 4 脚 | P1.6 | GPIO 输出高电平点亮 |

## CH552G SOP-16 关键引脚

| 物理脚 | 名称 | 本项目用途 |
| ---: | --- | --- |
| 1 | P3.2 | 预留 |
| 2 | P1.4 | 红灯控制 |
| 3 | P1.5 | 黄灯控制 |
| 4 | P1.6 | 绿灯控制 |
| 5 | P1.7 | 预留 |
| 6 | RST | 复位 / 测试焊盘 |
| 7 | P3.1 | 预留 |
| 8 | P3.0 | 预留 |
| 9 | P1.1 | 预留 |
| 10 | P3.3 | 预留 |
| 11 | P3.4 | 预留 |
| 12 | P3.6 / UDP | USB D+ |
| 13 | P3.7 / UDM | USB D- |
| 14 | GND | 地 |
| 15 | VCC | `5V` |
| 16 | V33 | 内部 USB 稳压输出，接 `100nF` 到地 |
