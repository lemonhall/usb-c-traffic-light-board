# 需求说明

## 产品目标

做一块 USB-C 控制的三灯红绿灯替换板。

## 必须满足

- 板子外形先按 `57mm x 14mm` 处理。
- 完全替换原板，不依赖原来的控制芯片。
- 使用 USB-C 同时完成供电和电脑控制。
- 三颗灯可以独立控制：
  - 红灯
  - 黄灯
  - 绿灯
- 最终装配尽量不飞线。
- 控制协议要能被电脑脚本直接调用。

## 应该满足

- 物料尽量选择国内常见、容易打样和焊接的器件。
- 原理图保持简单，方便手工排查。
- 关键电源和信号都留测试焊盘。

## 第一版不做

- 不做 Wi-Fi。
- 不做蓝牙。
- 不做电池充电。
- 不做 USB PD 协议协商。
- 不做大功率高亮 LED 驱动。
- 在精确尺寸出来前，不强行复刻全部缺口和孔位。

## 仍需补充的机械尺寸

- 三颗灯的中心坐标。
- 安装孔直径和中心坐标。
- 侧边缺口、倒角、圆角尺寸。
- USB-C 插座在外壳里的露出和避让空间。
