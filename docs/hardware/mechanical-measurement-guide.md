# 机械复核测量指南

目标是把原板外轮廓、两个螺丝孔、三颗灯中心、USB-C 后壳开口位置复刻到新板上。不要只靠手持尺子量轮廓，误差会很大；优先用可标定图片来描边。

## 推荐方法一：扫描仪

如果有平板扫描仪，这是最稳的方法。

1. 把原板正面朝下放在扫描仪玻璃上，旁边放一把钢尺或游标卡尺。
2. 扫描分辨率设为 `600dpi` 或更高。
3. 导出 `PNG`，不要用会压缩变形的截图。
4. 再把原板背面也扫一张，用来确认背面开关/USB-C 开口对应位置。
5. 文件命名建议：
   - `references/original-board-front-600dpi.png`
   - `references/original-board-back-600dpi.png`

有扫描图以后，可以用已知板长 `57mm` 或板宽 `14mm` 做比例标定，再在 KiCad 里描 `Edge.Cuts`。

## 推荐方法二：手机正射照片

如果没有扫描仪，用手机也可以，但要尽量减少透视变形。

1. 把原板放在白纸上，旁边放钢尺或游标卡尺，尺子要和板子同一平面。
2. 手机镜头尽量正对板子中心，不要斜拍。
3. 用 `2x` 或 `3x` 镜头，手机离远一点拍，能减少边缘畸变。
4. 打开网格线，让板子长边尽量和画面竖线平行。
5. 拍正面、背面各一张，最好再拍一张带外壳开口位置的照片。
6. 不要裁剪，不要加滤镜，直接发原图。

## 外轮廓怎么给

外轮廓不用逐段精确手量。最实用的做法是：

1. 先用扫描图或正射照片描出轮廓。
2. 用 `57mm x 14mm` 标定比例。
3. 再用 M1-M9 这些卡尺尺寸校正孔位和边界。
4. 斜角、侧边缺口、圆弧边按图片描边，优先保证能塞回外壳和螺丝孔对齐。

## USB-C 机械注意事项

当前 KiCad 工程里的 USB-C 封装是临时占位。已核查 `USB_C_Receptacle_G-Switch_GT-USB-7025` 的 KiCad 描述为 `right-angle, SMT`，它不能代表最终要从背面垂直伸出外壳的 USB-C 结构。

最终下单前必须补齐：

1. 确认实际采购的立式或背出 USB-C 母座型号。
2. 确认接口宽度约 `10mm` 能对齐原拨动开关开口。
3. 确认接口从板背面伸出的高度能替代原开关约 `10mm` 的伸出量。
4. 用该型号的数据手册重画或替换 KiCad footprint。
5. 加 3D 模型或机械标注，从侧面确认接口方向和外壳干涉。

## 拍照时额外注意

- 两个大圆孔是螺丝孔，孔中心距比外形小缺口更重要。
- 板边两侧的小半圆缺口可能是避让塑料柱或卡扣，需要保留大致位置。
- 顶部和底部的斜角不用过度追求 `0.1mm` 精度，能避开外壳干涉即可。
- 三颗灯中心要比丝印位置更重要，因为它们要对准红绿灯外壳的透光窗口。

## 你发给我的最小资料包

最少给这些就能继续画机械版：

1. 正面正射照片或扫描图一张。
2. 背面正射照片或扫描图一张。
3. `M1-M9` 的卡尺尺寸。
4. 如果能量，再补 `M10-M14`。

如果实在量不准，先给图片和 `M1-M5`，我可以先做一个近似轮廓版，再根据你试装反馈微调。
