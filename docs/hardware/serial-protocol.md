# USB 虚拟串口控制协议

传输方式：USB CDC 虚拟串口。

换行：支持 `\n` 或 `\r\n`。

编码：ASCII。

## 命令

| 命令 | 含义 |
| --- | --- |
| `R 1` | 打开红灯 |
| `R 0` | 关闭红灯 |
| `Y 1` | 打开黄灯 |
| `Y 0` | 关闭黄灯 |
| `G 1` | 打开绿灯 |
| `G 0` | 关闭绿灯 |
| `A 1` | 三颗灯全开 |
| `A 0` | 三颗灯全关 |
| `S r y g` | 一次设置红、黄、绿三个状态，每个值为 `0` 或 `1` |
| `?` | 查询当前状态 |

## 返回

| 返回 | 含义 |
| --- | --- |
| `OK` | 命令已接受 |
| `S R=0 Y=0 G=0` | 当前状态 |
| `ERR 1` | 命令或参数无法解析 |

## 上电状态

USB 复位后，三颗灯默认全灭。
